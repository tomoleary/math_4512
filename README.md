## Extra Credit

### Google Colab

To run the notebook in Google colab, open [the notebook](https://github.com/tomoleary/math_4512/blob/main/extra_credit/numerical_harmonic_functions.ipynb), and replace `github.com` with `githubtocolab.com`. Or just [click here!](https://githubtocolab.com/tomoleary/math_4512/blob/main/extra_credit/numerical_harmonic_functions.ipynb) This will then spawn a Google colab notebook. It will ask you to sign into a Google account to run it. 

This is the easiest way to run it, and will not require that you install any software on your computer; it can run fully in a browser. 

Once all of the questions have been answered inside of the notebook, you can choose File->Print->Save as PDF and download a pdf of the executed notebook. This can be handed in as a supplementary file for the extra credit submission.

### Alternative

Alternatively, you can run it locally using e.g., `conda` + `jupyter notebook` from the command line. But as this is substantial additional work, I do not encourage doing it unless you are already familiar with it.
